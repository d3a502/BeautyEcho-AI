# 妆声引擎 AI 方案书

## 项目全称

**妆声引擎 AI：面向逸仙集团的美妆社媒洞察与内容策略闭环系统**

英文名：

**BeautyEcho AI: A Social Beauty Intelligence and Content Strategy Loop for Yatsen Group**

## 一句话定位

妆声引擎 AI 是一个面向逸仙集团的美妆社媒洞察与内容策略闭环系统，帮助品牌从全网美妆声量中捕捉用户偏好、竞品机会和内容趋势，并生成可执行、可复盘、可迭代的传播策略。

## 命题理解

逸仙集团命题关注美妆内容在小红书、抖音、微博、B站等平台上的传播表现，核心不是“生成更多内容”，而是理解哪些话题、标题、内容形式和达人组合更容易引发用户关注、互动和讨论。

从业务角度看，社媒平台上的用户评论、达人测评、竞品爆文和投放反馈，都是品牌可以捕捉的“妆声”。如果这些声音不能沉淀为内容策略，就会停留在分散的运营经验中；如果能被 AI 结构化处理，就可以反哺产品卖点、达人 brief、平台内容策略和下一轮投放复盘。

## 解决方案

妆声引擎 AI 将社媒样本、产品信息、竞品案例和投放反馈统一沉淀到飞书多维表格中，并使用飞书 AI 对评论进行摘要、标签和策略初稿生成。系统通过多 Agent 流程完成趋势洞察、竞品分析、用户反馈聚类、产品-用户匹配、内容策略生成、合规审核和投放复盘。

## 核心流程

```text
美妆社媒样本
-> 飞书多维表格样本库
-> 飞书 AI 摘要/标签/初筛
-> 社媒趋势洞察
-> 竞品内容分析
-> 用户反馈聚类
-> 产品卖点与用户意愿匹配
-> 内容策略生成
-> 内容资产生成
-> 合规审核
-> 投放反馈复盘
-> 策略库更新
```

## 核心创新点

### 1. 从内容生成器升级为策略闭环

系统不是只生成小红书文案或抖音脚本，而是从趋势、竞品、用户反馈、产品卖点到复盘结果形成完整闭环。

### 2. 产品卖点与用户意愿匹配

内容策略生成前，系统先判断产品实际功效、价格带、适用人群与用户真实需求是否匹配，避免内容脱离产品事实。

### 3. 飞书 AI 与多 Agent 协作

飞书 AI 承担摘要、标签、策略初稿和复盘总结；多 Agent 分别负责趋势、竞品、用户反馈、产品匹配、内容生成和合规审核。

### 4. GitHub 研究证据包

报名阶段通过 GitHub 展示研究笔记、样例数据、飞书 AI 工作流、合规清单和轻量分析 notebook，证明方案具备继续落地的基础。

## 预期业务价值

- 提升社媒趋势发现效率。
- 降低内容策略试错成本。
- 提高产品卖点与用户需求匹配度。
- 帮助品牌沉淀可复用内容策略库。
- 为新品种草、达人 brief 和平台内容矩阵提供决策支持。

## 落地路径

报名阶段完成方案书和样例验证；入围后基于比赛飞书账号搭建飞书多维表格样本库，使用飞书 AI 完成评论摘要、痛点标签提取、策略初稿生成和复盘总结，并将样例分析结果接入演示流程。
