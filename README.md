# BeautyEcho AI

**面向逸仙集团的AI美妆社媒洞察与产品力表达闭环系统**

> 识别真实需求，让真正有产品力的好产品被看见、被理解、被选择。

## 仓库定位

本仓库用于展示 `BeautyEcho AI` 在报名阶段的研究基础与方案思路，不追求做出完整产品，而是回答三个关键问题：

1. 美妆社媒竞争正在发生什么变化。
2. 公开数据集与行业报告能否支撑前期洞察与建模验证。
3. 如何把洞察结果延伸为面向逸仙集团的产品优化、产品力表达与内容策略闭环。

## 项目主线

我们将项目核心聚焦在“智能 AI 洞察与分析”能力上。系统不只是生成更多内容，而是先识别消费者对成分、配方、妆效、功效表达和竞品差异的真实关注，再将这些洞察反向用于：

- 产品优化与产品力提炼；
- 内容策略制定与产品力表达；
- 投放复盘与策略库更新。

## 当前仓库内容

```text
research/     行业背景、公开数据集说明、参考资料
proposal/     开题报告、项目方案概览
analysis/     基于公开美妆数据的 EDA 与评分预测基线
outputs/      研究结论摘要与脚本输出目录
data/         数据放置说明（不直接上传第三方完整数据）
assets/       架构图与展示素材
```

## 推荐阅读顺序

1. `proposal/开题报告.md`
2. `research/行业背景研究.md`
3. `research/公开数据集说明.md`
4. `outputs/公开数据研究结论摘要.md`

## 本地运行方式

本地准备公开美妆数据后可运行基础研究脚本：

```bash
pip install -r requirements.txt
python analysis/01_公开数据EDA.py
python analysis/02_公开数据评分预测.py
```

脚本会优先从以下位置读取数据：

1. `data/skincare_df.csv`
2. 仓库同级目录下的 `archive/skincare_df.csv`

## 数据与合规说明

仓库不直接上传第三方完整数据集，不包含企业内部数据，也未接入真实平台全量社媒数据。当前研究围绕公开数据集与行业报告做方法验证，并在后续方案中结合飞书智能伙伴、多维表格等能力构建可落地的协同闭环。
