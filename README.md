# 妆声引擎 AI / BeautyEcho AI

**面向逸仙集团的美妆社媒洞察与内容策略闭环系统**

> 听见用户回响，看见美妆趋势。

## 项目概述

妆声引擎 AI（BeautyEcho AI）是一个面向美妆品牌的社媒洞察与内容策略闭环系统。项目聚焦逸仙集团命题“基于 AI 的美妆社交媒体传播分析与内容策略优化”，将小红书、抖音、微博、B站等平台中的用户反馈、竞品内容、达人表达和互动数据视为品牌需要持续捕捉的“妆声”，并通过飞书 AI、多 Agent 流程和轻量机器学习分析，将分散的社媒信号转化为可执行、可复盘、可迭代的内容策略。

## 为什么做这个项目

美妆内容传播已经从“发布种草内容”变成“持续理解用户、校准产品卖点、生成内容策略并快速复盘”的闭环过程。逸仙集团拥有多品牌矩阵和产品研发能力，但社媒趋势、竞品打法、用户反馈和产品卖点往往分散在不同渠道中。妆声引擎 AI 希望帮助品牌把这些信号连接起来，形成内容决策能力。

## 系统闭环

```text
美妆社媒样本
-> 飞书多维表格样本库
-> 飞书 AI 摘要/标签/初筛
-> 社媒趋势洞察
-> 竞品内容分析
-> 用户反馈聚类
-> 产品卖点与用户意愿匹配
-> 内容策略生成
-> 内容资产生成
-> 合规审核
-> 投放反馈复盘
-> 策略库更新
```

## 核心模块

| 模块 | 功能 |
|---|---|
| 妆声雷达 | 捕捉平台热词、热门标题、内容形式和达人类型 |
| 趋势声纹 | 提炼审美趋势、妆容场景和功效偏好 |
| 竞品回响 | 拆解竞品爆款内容和用户讨论 |
| 用户共鸣 | 聚类真实评论、购买顾虑和使用痛点 |
| 产品映射 | 匹配产品卖点、用户意愿和内容场景 |
| 灵感工坊 | 生成小红书笔记、抖音脚本、达人 brief |
| 合规校音 | 审核功效宣称、品牌调性和风险表达 |
| 复盘回路 | 根据投放反馈更新策略库 |

## 飞书 AI 使用方案

完整参赛作品阶段将至少使用一项飞书 AI 产品能力：

- 使用飞书多维表格作为社媒样本、产品信息和投放反馈的数据底座。
- 使用飞书 AI 对用户评论进行摘要、情绪判断和痛点标签提取。
- 使用飞书 AI 辅助生成内容策略初稿和投放复盘摘要。
- Python/机器学习分析仅作为趋势聚类、竞品分析和互动预测的辅助方法。

报名阶段先完成工作流设计和样例数据验证；入围并激活比赛飞书账号后，将在飞书多维表格与飞书 AI 中完成实际部署演示。
