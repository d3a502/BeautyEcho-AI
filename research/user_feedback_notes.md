# 用户反馈笔记

## 用户反馈分类

### 1. 功效与体验

- 持妆。
- 遮瑕。
- 卡粉。
- 拔干。
- 暗沉。
- 妆感自然。

### 2. 肤质与场景

- 油皮。
- 干皮。
- 混油。
- 通勤。
- 夏季。
- 约会。
- 学生党。

### 3. 购买顾虑

- 价格。
- 色号选择。
- 成分安全。
- 真实效果是否如宣传。
- 达人推荐是否可信。

### 4. 内容偏好

- 前后对比。
- 妆教教程。
- 真实测评。
- 成分解释。
- 平价替代。
- 场景化种草。

## 可转化为策略的洞察

- 用户不只关注产品功效，也关注“适合我吗”。
- 内容策略需要从产品卖点转向肤质、场景和人群表达。
- 达人 brief 应突出真实体验和可验证效果。
